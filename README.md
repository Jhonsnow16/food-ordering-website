# QuickFood - Delivery Service Website

A simple food delivery service website built with HTML and CSS.

## Project Files

- **index.html** - Main HTML file with the website structure
- **style.css** - Stylesheet for the website design
- **images/** - Directory containing logo and food images

## Features

- Navigation bar with menu items (Home, Menu, About, Restaurants, Contact)
- Shopping cart display
- Business contact information
- Responsive design

## Getting Started

1. Open `index.html` in a web browser
2. The website displays a navigation bar with the QuickFood logo and links to different sections
3. Cart information is displayed with total price and item count

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No additional dependencies required
